This project is a checkout sample 
I use vue3 - Vuetify for UI - Pinia for state management 


### Setup

```shell
npm install
```

Launch the APIs this will run on port 3001

```
npm run api
```

This project uses vite to run the dev server execute

```
npm run dev
```
